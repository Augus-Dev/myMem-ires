# myMemóires - Landing Page

A landing page for "myMémoires" - a digital platform that converts photo galleries into structured albums using customizable templates.
**[View Live Project](https://augus-dev.github.io/myMem-ires/)**

## Future goal
myMémoires is a web application that provides template-based photo album creation, offering an alternative to standard gallery storage with organized layout options.

## Tech Stack

- **Frontend**: HTML5, CSS3
- **Design**: Responsive layout, custom UI components
- **Deployment**: GitHub Pages
